map vs forEach

javascript 2016-05-24

二者传入参数的规则完全相同
都是数组对象的方法，他们有什么区别呢？

可以直接阅读stackoverflow的答案

结论：map方法是forEach方法的超集。
二者都对数组对象的元素进行了遍历，但是map对象会将callback函数return的值存入新的数组作为返回值。
而forEach方法只进行了遍历调用callback，并没有返回值。

map方法占用资源多，用于对数组进行转换操作，callback需要返回值

var a = [1,2,3,4,5];
var b = a.map(function(value){
    return value*value;
})
//b = [1,4,9,16,25];

forEach方法进行遍历个元素,不需要生成对应数组，callback不需要返回值

var a = [1,2,3,4,5];
var sum = 0;
a.forEach(function(value){
    sum += value;
})
//sum = 15
